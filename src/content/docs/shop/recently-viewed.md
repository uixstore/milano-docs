The Recently Viewed section shows customers products they have previously looked at. It helps bring visitors back to items they were interested in.

## How it works

Milano tracks product views independently using a cookie. Each visitor sees their own recently viewed products — no login required. The cookie stores up to 15 products.

## Where it appears

Recently viewed products can appear in two places:

- **Single product pages** — shown below the product content as a carousel.
- **Dedicated page** — create a page with the `[recently_viewed]` shortcode to show a full recently viewed section.

## Configure the section

1. Open **Appearance → Customize → Product Page → Recently Viewed**.
2. Adjust the settings:

| Setting     | Description                                      |
| ----------- | ------------------------------------------------ |
| **Enable**  | Turn the section on or off.                      |
| **Columns** | Number of product columns in the carousel (2–6). |
| **Limit**   | Maximum number of products to show (1–12).       |

## Create a Recently Viewed page

1. Go to **Pages → Add New**.
2. Give the page a title (for example, "Recently Viewed").
3. Add the shortcode `[recently_viewed]` in the content.
4. Publish the page.

Customers can then visit this page to see all the products they recently browsed.

:::tip
Add a link to your Recently Viewed page in your footer or navigation menu so customers can find it easily.
:::
