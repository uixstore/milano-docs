## What is a purchase code?

Your purchase code (also called a license key) is a unique identifier for your copy of Milano. You find it in your Envato (ThemeForest) account under **Downloads → Milano → License**.

## Why do I need to register my purchase code?

Registering your purchase code unlocks:

- Automatic theme updates from within WordPress
- Access to demo content imports
- Installation of bundled premium plugins
- Eligibility for support from the UIX Themes team

## Where do I enter my purchase code?

Go to **Milano → Dashboard → License** in your WordPress admin. Paste your purchase code in the field and click **Register**.

## Can I use one purchase code on multiple sites?

No. Each purchase code is valid for one live website. You can use it on a staging or development copy of the same site, but not on a second live domain.

If you need Milano on multiple sites, buy a separate license for each one.

## What happens if I lose my purchase code?

Log in to your Envato account and go to **Downloads**. Your purchase code appears next to Milano in the list. You can also find it in the purchase confirmation email from Envato.

## Can I transfer my license to another person?

You can transfer the license through Envato's official transfer process. Contact Envato support to initiate a transfer. The new owner receives the purchase code and support eligibility.

## My purchase code says "already registered"

This happens when the code was entered on another site. If you moved your site to a new domain:

1. Go to **Milano → Dashboard → License** on the old site (if you still have access).
2. Click **Deactivate** or **Remove license**.
3. Enter the code on the new site.

If you cannot access the old site, contact UIX Themes support with your purchase details and we will release the license.

## Does my license expire?

Your license includes one year of support and updates from the purchase date. After one year:

- The theme continues to work on your site.
- You can still use all features.
- You lose access to new updates and support until you renew.

Renew your license through your Envato account to restore update and support access.

## What does the license cover?

Your license covers:

- Use of Milano on one live website
- Theme updates for one year
- Support for installation, configuration, and bug fixes for one year

Your license does not cover:

- Custom development work
- Third-party plugin support
- Hosting or server configuration
- Design changes or customizations

## Troubleshooting

**Problem:** My purchase code is rejected as invalid.
**Fix:** Copy the code directly from your Envato Downloads page. Do not type it manually — it is easy to mistake similar characters.

**Problem:** I registered my code but demo import still does not work.
**Fix:** Make sure all required plugins are installed and active. Go to **Milano → Dashboard** and check the plugin status. Install any missing plugins before importing demos.
