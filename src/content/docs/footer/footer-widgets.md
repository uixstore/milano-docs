Each footer row you enable in the footer builder creates a corresponding widget area in **Appearance → Widgets**. You can add any WordPress widget to these areas.

Footer widget areas are named after their row: **Footer Top**, **Footer Middle**, and **Footer Bottom**.

## Add a widget to the footer

1. Go to **Appearance → Widgets**.
2. Find the widget area for the footer row you want to edit.
3. Click the **+** button to add a widget.
4. Search or browse for the widget type you want.
5. Configure the widget settings and click **Update**.

## Common footer widgets

| Widget                 | What it does                                        |
| ---------------------- | --------------------------------------------------- |
| **Recent Posts**       | Shows your latest blog posts with dates.            |
| **Pages**              | Lists your site pages as links.                     |
| **Product Categories** | Lists your WooCommerce categories.                  |
| **Custom HTML**        | Any HTML content — newsletter forms, badges, etc.   |
| **Navigation Menu**    | Display a specific menu in the footer.              |
| **Social Icons**       | Links to social media profiles with platform icons. |
