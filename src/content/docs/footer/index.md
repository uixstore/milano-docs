The Milano footer is made up of up to three rows stacked vertically. Each row holds different content, and you control what goes in them with the footer builder.

## Row structure

| Row               | What it contains                                                                           |
| ----------------- | ------------------------------------------------------------------------------------------ |
| **Footer top**    | The top row of the footer. Use it for extra widgets, a newsletter signup, or social links. |
| **Footer middle** | The main footer row. Use it for widget columns, menus, and contact information.            |
| **Footer bottom** | The bottom row. Use it for copyright text, payment icons, and secondary footer links.      |

## What you can do

- [Open the footer builder](./footer-builder/) and start adding elements
- [Add widgets to footer areas](./footer-widgets/) — recent posts, social links, newsletter forms
- Configure copyright text, layout, and colors in **Appearance → Customize → Footer**

## Related settings

The **Footer** panel in the Customizer controls layout, rows, columns, colors, and copyright text. See [Footer settings](../theme-settings/footer/).
