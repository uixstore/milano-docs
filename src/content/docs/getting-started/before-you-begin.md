Read this page before installing Milano. It covers what you need, what you get in the download, and how long setup takes.

## What you will need

- **WordPress 5.9 or higher** — check your version under **Dashboard → Updates** in your WordPress admin.
- **PHP 7.4 or higher** — ask your host or check under **Tools → Site Health → Info → Server**.
- **A working WordPress installation** — Milano is a theme, not a standalone app. You need WordPress running first.
- **Admin access** to your WordPress dashboard.
- **A supported browser** — Chrome, Firefox, Safari, or Edge, latest stable version.
- **The Milano theme zip** — downloaded from your ThemeForest account.

:::tip
If you are not sure about your PHP version, most modern hosts run PHP 8.0 or higher. You can check and change it from your hosting control panel.
:::

## What is in your download

Go to your ThemeForest **Downloads** page and click **Download** next to Milano. You will see two options:

| File                             | What it contains                                                   |
| -------------------------------- | ------------------------------------------------------------------ |
| Installable WordPress theme only | The Milano theme zip — this is what you upload to WordPress.       |
| All files and documentation      | Everything above, plus the child theme, demo content, and license. |

For a fresh install, download **Installable WordPress theme only**. If you also want the child theme or demo content files, grab **All files and documentation**.

When you download the full package and extract it, you will find:

- `milano.zip` — the main theme file.
- `milano-child.zip` — the child theme (optional; see [Install the child theme](../install-the-child-theme/)).
- License file — your purchase proof. Keep this safe.
- Demo content files — used by Milano's demo importer.

## Estimated time

From a fresh WordPress install to a working Milano site takes about 30 minutes:

- Install and activate Milano — 5 minutes.
- Register your purchase code — 2 minutes.
- Install required plugins — 5 minutes.
- Import a demo or set up from scratch — 15–20 minutes.
