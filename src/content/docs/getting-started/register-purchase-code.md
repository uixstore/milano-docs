After installing Milano, register your purchase code. This unlocks theme updates, demo imports, bundled premium plugins, and support from the UIX Themes team.

:::caution
You must register your purchase code before Milano can check for updates.
:::

## Find your purchase code

1. Log in to your ThemeForest account.
2. Go to your **Downloads** page.
3. Find Milano in your list and click **Download** next to it.
4. Select **License certificate** from the dropdown. This downloads a text or PDF file.
5. Open the file and copy the **Purchase Code** — it looks like a long string of letters, numbers, and dashes.

![ThemeForest Downloads page with the License certificate download option highlighted](../../../assets/shared/placeholder.svg)

_ThemeForest Downloads page with the "License certificate" option highlighted in the dropdown next to Milano._

## Enter your purchase code in Milano

1. Go to **Milano → Dashboard → License** in your WordPress admin.
2. Paste your purchase code into the **License Key** field.
3. Click **Activate License**.

![Milano License screen with the License Key field and Activate License button highlighted](../../../assets/shared/placeholder.svg)

_Milano → Dashboard → License screen with the license key input field and "Activate License" button highlighted._

You will see a confirmation message when the license is active. Milano can now check for updates.

## Troubleshooting

**Problem:** "Invalid purchase code."
**Fix:** Make sure you copied the full code with no extra spaces. The code comes from your License certificate file — double-check that you copied the right string.

**Problem:** "This purchase code is already registered."
**Fix:** Each purchase code works on one live site. If you used it on a test or staging site, deactivate it there first under **Milano → Dashboard → License**, then activate it on your live site. If you bought a second license, use that code instead.

**Problem:** "Could not connect to the license server."
**Fix:** This is usually a temporary server issue. Wait a minute and try again. If it keeps failing, check that your server can reach external APIs — some hosts block outgoing connections by default.
