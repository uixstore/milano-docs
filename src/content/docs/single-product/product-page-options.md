Control the layout, gallery, tabs, and product sections on single product pages.

All settings live under **Appearance → Customize → Product Page**.

## Gallery layout

Set the image gallery style and how customers interact with product images.

### Layout options

| Option      | Description                                           |
| ----------- | ----------------------------------------------------- |
| **Slider**  | A horizontal carousel with thumbnails and navigation. |
| **Stacked** | Images stacked vertically below each other.           |
| **Grid**    | A grid of images in rows and columns.                 |
| **Masonry** | A grid where images keep their original aspect ratio. |

### Thumbnail position (Slider mode)

Choose where thumbnails appear relative to the main image: **Bottom**, **Left**, **Right**, or **Hidden**.

### Zoom

Turn on zoom so customers can inspect product details up close. Three styles are available:

- **Inner** — Zoomed image overlays the main image in place.
- **Magnifier** — A magnifying glass effect follows the cursor.
- **External** — Zoomed image appears in a separate side pane.

### Lightbox

Turn on the lightbox to open a full-screen image viewer (PhotoSwipe) when customers click the main image.

### Fine-tuning

| Setting           | Range                       | Affects                              |
| ----------------- | --------------------------- | ------------------------------------ |
| **Border radius** | 0–20 px                     | Rounded corners on gallery images.   |
| **Gap**           | 0–30 px                     | Space between gallery images.        |
| **Aspect ratio**  | Natural, 1:1, 3:4, 4:3, 2:3 | Forces images to a consistent ratio. |

## Tabs

WooCommerce product tabs (Description, Additional Information, Reviews) are shown below the product summary. You can turn them on or off and choose a style:

- **Horizontal Tabs** — Default tabbed layout with clickable labels.
- **Accordion** — Collapsible panels stacked vertically.
- **Vertical Tabs** — Labels on the left, content on the right.

When using accordion mode, choose the initial state: **Open first**, **Open all**, or **Close all**.

## Sticky summary

Turn on the sticky summary to keep the product name, price, and add-to-cart button visible as customers scroll down the page.

| Setting                | Description                                         |
| ---------------------- | --------------------------------------------------- |
| **Sticky summary**     | Pin the summary section on scroll. On by default.   |
| **Sticky add to cart** | Show a sticky add-to-cart bar at the screen bottom. |
| **Content spacing**    | Gap between the gallery and summary, 0–100 px.      |

## Page header

Control what appears above the product content:

- **Page header** — Show or hide the header area above the product.
- **Breadcrumbs** — Display breadcrumb navigation in the header.
- **Product navigation** — Show previous/next product links in the header.

## Summary component visibility

Toggle individual elements in the product summary:

- Product overline
- Excerpt
- Product meta (SKU, categories, tags)
- Share buttons
- Safe checkout icons
- Product info items

## Related and upsell products

Both sections share the same controls:

| Setting      | Range         | Description                     |
| ------------ | ------------- | ------------------------------- |
| **Enable**   | On / Off      | Show or hide the section.       |
| **Count**    | 1–12 products | Number of products to show.     |
| **Columns**  | 2–6 columns   | Layout grid width.              |
| **Carousel** | On / Off      | Switch from grid to a carousel. |

## Recently viewed products

Show a carousel of products the visitor has recently viewed. Configure the number of columns and the product limit.
