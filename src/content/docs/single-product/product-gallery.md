Control how product images appear on single product pages — layout style, thumbnail position, zoom behavior, and lightbox.

All settings live under **Appearance → Customize → Product Page → Gallery**.

## Gallery layout

Choose how customers browse product images.

| Option      | Description                                           |
| ----------- | ----------------------------------------------------- |
| **Slider**  | A horizontal carousel with thumbnails and navigation. |
| **Stacked** | Images stacked vertically below each other.           |
| **Grid**    | A grid of images in rows and columns.                 |
| **Masonry** | A grid where images keep their original aspect ratio. |

## Thumbnail position

When using **Slider** layout, choose where thumbnails appear relative to the main image:

- **Bottom** — thumbnails below the main image.
- **Left** — thumbnails in a column on the left.
- **Right** — thumbnails in a column on the right.
- **Hidden** — no thumbnails shown.

## Zoom

Turn on zoom so customers can inspect product details up close.

Three styles are available:

- **Inner** — zoomed image overlays the main image in place.
- **Magnifier** — a magnifying glass effect follows the cursor.
- **External** — zoomed image appears in a separate side pane.

## Lightbox

Turn on the lightbox to open a full-screen image viewer when customers click the main image. The lightbox uses PhotoSwipe and supports pinch-to-zoom on mobile.

## Fine-tuning

<!-- markdownlint-disable MD060 -->

| Setting           | Range                       | Affects                              |
| ----------------- | --------------------------- | ------------------------------------ |
| **Border radius** | 0–20 px                     | Rounded corners on gallery images.   |
| **Gap**           | 0–30 px                     | Space between gallery images.        |
| **Aspect ratio**  | Natural, 1:1, 3:4, 4:3, 2:3 | Forces images to a consistent ratio. |
