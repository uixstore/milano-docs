You can change the site's layout dimensions from **Appearance → Customize → Site Layout**.

## Container widths

Milano uses three container sizes depending on the page or component:

| Setting                    | Default | Range         | What it controls                     |
| -------------------------- | ------- | ------------- | ------------------------------------ |
| **Container Width**        | 1200px  | 960 – 1600px  | Standard content container           |
| **Wide Container Width**   | 1410px  | 1200 – 1920px | Full-width sections and wide layouts |
| **Narrow Container Width** | 840px   | 600 – 1200px  | Blog posts and narrow content        |

## Rounded corners

The **Rounded Radius** setting controls the corner radius on cards, boxes, and UI elements. Default is 10px, range is 0 – 50px. Set it to 0 for sharp corners.
