The **Mobile** panel in the Customizer controls the mobile experience — the off-canvas menu and the bottom navigation bar.

## What you can set

- **Mobile Menu** — menu elements (menu, phone, social, login, recently viewed, wishlist), category display, language and currency switchers, link behavior, header and background colors
- **Bottom Bar** — enable or disable, bar items (home, shop, cart, account, wishlist)

For detailed setup instructions, see [Navigation](../navigation/).
