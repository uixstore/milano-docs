The **Header** panel in the Customizer controls everything above your page content: the logo, navigation menus, cart and account icons, search bar, and contact information.

## What you can set

- **Header** — sticky behavior (always, on scroll up, or disabled), container width, border
- **Logo** — main and alternative logos, logo size per device
- **Menus** — menu font, dropdown icons, submenu animation, custom menu selection
- **Categories** — category navigation bar with images, nesting depth, empty category visibility
- **Cart** — icon or text display, drawer or link behavior (requires WooCommerce)
- **Account** — icon or text display, modal or link behavior (requires WooCommerce)
- **Search** — icon or text trigger, AJAX product search, trending keywords, default products
- **Phone** — display mode and business hours
- **Email** — display settings
- **Top Bar** — announcements, social links, language and currency switchers

For detailed setup instructions, see [Header builder](../header/header-builder/).
