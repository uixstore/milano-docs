title: Custom CSS

---

You can add custom CSS from **Appearance → Customize → Additional CSS**.

The editor includes syntax highlighting and real-time preview. Changes apply immediately when you publish.

:::tip
Custom CSS is saved in the database and survives theme updates. For more advanced customization, [create a child theme](https://developer.wordpress.org/themes/advanced-topics/child-themes/).
:::
