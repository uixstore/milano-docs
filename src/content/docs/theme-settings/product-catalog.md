The **Product Catalog** panel in the Customizer controls how your product listings look — the shop page, category pages, product cards, and category cards.

## What you can set

- **Shop Layout** — filter position, toolbar elements (count, columns, ordering), pagination type (numeric, load more, infinite scroll)
- **Shop Page Header** — enable or disable, background image, breadcrumbs, height, and colors
- **Top Categories** — carousel of category thumbnails above products, image ratio, font size
- **Product Card** — layout style, thumbnail hover effect, rating, wishlist, compare, quick view, swatches
- **Category Card** — layout style, image ratio, product count visibility

For detailed setup instructions, see [Shop layout](../shop/).
