You can change the site's fonts from **Appearance → Customize → Global**, under the **Typography** section.

## Body font

The **Main Font** setting controls the font used for body text, paragraphs, and most page content. You can set:

- **Font family** — choose from available Google Fonts or system fonts
- **Weight** — regular, medium, bold, or another available weight
- **Size** — default is 15px
- **Line height** — default is 1.714
- **Letter spacing** — default is 0

## Heading font

The **Heading Font** setting controls all headings (H1–H6). You can set:

- **Font family**
- **Weight** — default is 600 (semibold)
- **Line height** — default is 1.21
- **Letter spacing** — default is -0.034em

## Subheading font

The **Subheading Font** setting controls the weight of subheading elements. Default weight is 500 (medium).

Not seeing the font you want? Milano syncs with Google Fonts — type the font name in the family field to search.
