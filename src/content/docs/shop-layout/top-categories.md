Top categories show a carousel of your top-level product categories above the product grid. Customers can click a category to jump directly to it.

## Turn on Top Categories

1. Open **Appearance → Customize → Product Catalog → Top Categories**.
2. Toggle **Enable** to turn on the carousel.

## Display settings

| Setting        | Description                                    |
| -------------- | ---------------------------------------------- |
| **Display on** | Show on all catalog pages or shop page only.   |
| **Limit**      | Number of categories to show (1–24).           |
| **Columns**    | Number of columns in the carousel (1–12).      |
| **Order by**   | Sort by default order, name, or product count. |

## Image ratio

Control the aspect ratio of category thumbnails:

- **Adapt to image** — no cropping
- **Square** (1:1)
- **Portrait** (3:4)
- **Landscape** (4:3)
- **Classic** (2:3)
- **Custom** — enter a CSS aspect-ratio value

## Text alignment

Align category names to the **Left**, **Center**, or **Right** under each thumbnail.

## Font size

Adjust the category name font size with a slider (8–100 px).

## How it works

The carousel uses a slider component. Customers can scroll horizontally through categories on touch devices or with arrow controls on desktop.

:::tip
Keep the category limit between 6 and 12 for the best visual balance. Too many categories can overwhelm the page.
:::
