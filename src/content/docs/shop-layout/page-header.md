The page header is the banner area that appears above your shop and product archive pages. It typically shows the page title and breadcrumb navigation.

## Turn on or off

1. Open **Appearance → Customize → Product Catalog → Page Header**.
2. Toggle **Enable** to show or hide the header on shop pages.

## Background image

Set a background image for the page header. This image appears behind the title and breadcrumbs.

- Upload an image in the **Background Image** setting.
- Turn on **Overlay** to add a semi-transparent color layer over the image.

## Per-category headers

Each product category can have its own header image. To set one:

1. Go to **Products → Categories**.
2. Edit a category.
3. Upload an image in the **Header Image** field.

When a customer views that category archive, the category-specific image replaces the default.

## Breadcrumbs

Turn on **Breadcrumbs** to show navigation links in the header. Breadcrumbs help customers understand where they are in your store — for example, **Home → Shop → T-Shirts**.

## Height

Adjust the header height with the **Height** slider (100–500 px). A taller header gives more space for background images.

## Colors

Set custom colors for the header area:

- **Background color** — solid color behind the image (or instead of it).
- **Text color** — color for the title and breadcrumb text.
