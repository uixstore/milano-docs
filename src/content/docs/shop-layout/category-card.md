Category cards control how product category thumbnails appear in the top categories carousel and category archive pages.

All settings live under **Appearance → Customize → Product Catalog → Category Card**.

## Card style

Choose a card style for category thumbnails:

| Style       | Description                                    |
| ----------- | ---------------------------------------------- |
| **Default** | Simple thumbnail with the category name below. |
| **Card**    | Elevated card with shadow and hover effects.   |
| **Modern**  | Minimal style with overlay text on the image.  |

## Product count

Turn on **Show Count** to display the number of products in each category below the thumbnail.

## Image ratio

Control the aspect ratio of category thumbnails:

- **Adapt to image** — no cropping, original ratio preserved
- **Square** (1:1)
- **Portrait** (3:4)
- **Landscape** (4:3)
- **Classic** (2:3)
- **Custom** — enter a CSS aspect-ratio value

:::tip
Match the category card image ratio to your product card ratio for a consistent look across your store.
:::
