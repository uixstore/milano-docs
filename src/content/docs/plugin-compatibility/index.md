Milano is built to work alongside the WordPress plugins you already use. Each page in this section covers a specific plugin and explains how it interacts with the theme.

## In this section

- [Yoast SEO](./yoast-seo/) — Use Milano with Yoast SEO for search engine optimization.
- [Rank Math](./rank-math/) — Use Milano with Rank Math for search engine optimization.
- [WPML](./wpml/) — Create a multilingual store with WPML.
- [Polylang](./polylang/) — Create a multilingual store with Polylang.
- [Elementor](./elementor/) — Build pages with Elementor using Milano's theme builder locations.
- [WooCommerce](./woocommerce/) — Milano's deep integration with WooCommerce.

:::note
Milano works with most WordPress plugins out of the box. If a plugin is not listed here, it likely works without extra configuration. Contact support if you run into conflicts.
:::
