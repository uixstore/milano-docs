The phone element displays your business phone number.

## Where to find it

Header builder → **+ Add Element** → **Phone**

## Settings

| Setting            | Description                                    |
| ------------------ | ---------------------------------------------- |
| **Phone number**   | Your contact number.                           |
| **Display mode**   | Icon only, text only, or both.                 |
| **Business hours** | Optional hours displayed alongside the number. |
