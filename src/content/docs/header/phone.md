The phone element displays your business phone number.

:::note[Global source]
The phone number and business hours are set in **Customize → Store → Contact Information**. Changes there will update the phone element automatically.
:::

## Where to find it

Header builder → **+ Add Element** → **Phone**

## Settings

| Setting                 | Description                                    |
| ----------------------- | ---------------------------------------------- |
| **Display mode**        | Icon only, text only, or both.                 |
| **Show business hours** | Optional hours displayed alongside the number. |
| **Phone Icon**          | Select the phone icon for header element       |
