The custom HTML element lets you insert your own HTML into the header.

## Where to find it

Header builder → **+ Add Element** → **Custom HTML**

## Settings

| Setting          | Description                            |
| ---------------- | -------------------------------------- |
| **HTML content** | Paste your HTML code.                  |
| **CSS class**    | Optional custom CSS class for styling. |
