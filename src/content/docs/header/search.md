The search element lets visitors search your site from the header.

## Where to find it

Header builder → **+ Add Element** → **Search**

## Settings

| Setting               | Description                                                    |
| --------------------- | -------------------------------------------------------------- |
| **Trigger type**      | Display as an icon, a text button, or both.                    |
| **Placeholder text**  | The placeholder shown inside the search field.                 |
| **AJAX search**       | Turn on to show product and post results as the visitor types. |
| **Trending keywords** | Show popular search terms below the search field.              |
| **Default products**  | Show placeholder products when the search field is empty.      |
