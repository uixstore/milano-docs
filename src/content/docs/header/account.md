The account element shows a login link or account icon for customer accounts.

## Where to find it

Header builder → **+ Add Element** → **Account**

## Settings

| Setting          | Description                                         |
| ---------------- | --------------------------------------------------- |
| **Display type** | Icon, text link, or both.                           |
| **Interaction**  | Open a login modal, or link to the My Account page. |

## Prerequisites

WooCommerce must be installed and active.
