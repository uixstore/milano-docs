The primary menu element shows your navigation menu in the header.

## Where to find it

Header builder → **+ Add Element** → **Primary Menu**

## Settings

| Setting               | Description                                                           |
| --------------------- | --------------------------------------------------------------------- |
| **Menu selection**    | Choose which WordPress menu to display in this slot.                  |
| **Font size**         | Base font size for menu links.                                        |
| **Dropdown icons**    | Show or hide arrows on parent menu items that have submenus.          |
| **Submenu animation** | How submenus appear — slide down, fade in, or instant (no animation). |

## Prerequisites

A menu must be created under **Appearance → Menus** and assigned to a location.
