The email element displays your email address.

## Where to find it

Header builder → **+ Add Element** → **Email**

## Settings

| Setting           | Description                    |
| ----------------- | ------------------------------ |
| **Email address** | The email to display.          |
| **Display mode**  | Icon only, text only, or both. |
