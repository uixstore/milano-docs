The cart element displays the shopping cart with an item count badge.

## Where to find it

Header builder → **+ Add Element** → **Cart**

## Settings

| Setting           | Description                                       |
| ----------------- | ------------------------------------------------- |
| **Display type**  | Icon with count badge, or text link.              |
| **Cart behavior** | Link to the cart page, or open a slide-in drawer. |
| **Icon style**    | Choose from available cart icon styles.           |

## Prerequisites

WooCommerce must be installed and active.
