The Product Tabs module gives you full control over the tabs on WooCommerce product pages. You can reorder and hide the default tabs (Description, Additional Information, Reviews) and create custom tabs.

## Turn on Product Tabs

1. Go to **Milano Dashboard → Modules**.
2. Find **Product Tabs** and turn the toggle on.

## Manage default tabs

1. Go to **Product Tabs → All Product Tabs**.
2. You'll see the default tabs listed: **Description**, **Additional Information**, and **Reviews**.
3. Drag to reorder them.
4. Use the toggle to hide any tab you don't want to show.
5. Click **Update**.

## Create a custom tab

1. Go to **Product Tabs → Add New**.
2. Give the tab a **Title** (shown to customers).
3. Enter the **Content** — text, HTML, or shortcodes.
4. Set **Visibility**:
   - **Global** — Shows on all products.
   - **Category** — Shows only in selected categories.
   - **Product-specific** — Shows only on selected products.
5. **Publish** the tab.

The tab appears in the order you set on the **All Product Tabs** screen.

## Assign a tab to a specific product

1. Edit a product.
2. In the **Product Data** meta box, find the **Product Tabs** section.
3. Select which custom tabs to show on this product.
4. **Update** the product.

## Requirements

- WooCommerce must be installed and active.
