The Size Guide module lets you create size guide charts and assign them to products. Customers see a **Size Guide** link on the product page that opens the chart in a modal.

## Turn on Size Guide

1. Go to **Milano Dashboard → Modules**.
2. Find **Size Guide** and turn the toggle on.

## Create a size guide

1. Go to **Size Guides → Add New**.
2. Give the guide a name (internal, for your reference).
3. Build the size chart using the table editor:
   - Add columns for each measurement (for example, Size, Bust, Waist, Hip).
   - Add rows for each size option.
4. **Assign the guide to categories** in the **Categories** meta box — products in those categories will show the guide automatically.
5. **Publish** the guide.

You can also assign a specific size guide to an individual product from the product edit screen.

## Assign to a product

1. Edit a product.
2. Scroll to the **Product Data** meta box.
3. Find the **Size Guide** dropdown and select the guide you want.
4. Update the product.

A product-level assignment overrides any category-based assignment.

## Display on the frontend

Customers see a **Size Guide** link on the product page. Clicking it opens the chart in a modal overlay with the measurements you entered.

## Requirements

- WooCommerce must be installed and active.
