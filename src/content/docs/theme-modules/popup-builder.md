The Popup Builder lets you create popups using Elementor. You can show them based on visitor behavior — after a delay, when someone tries to leave, after scrolling, or on a click. You control how often each visitor sees the popup.

## Turn on Popup Builder

1. Go to **Milano Dashboard → Modules**.
2. Find **Popup Builder** and turn the toggle on.

## Create a popup

1. Go to **Milano → Popups → Add New**.
2. Give your popup a name.
3. **Edit with Elementor** and design the popup content. You can use any Elementor widget, plus the dedicated **Popup Close Button** widget.
4. **Publish** the popup.

## Configure triggers and behavior

Open the popup for editing. In the Elementor editor, find the **Popup Settings** panel:

**Triggers** — When the popup appears:

- **On page load** — Shows immediately or after a delay.
- **On exit intent** — Shows when the cursor moves toward the browser's address bar.
- **On scroll** — Shows after the visitor scrolls a percentage of the page.
- **On click** — Shows when a specific element is clicked.
- **On timeout** — Shows after a set number of seconds.

**Frequency** — How often each visitor sees it:

- **Once per session** — Shows once per browser session.
- **Once per day** — Shows once every 24 hours.
- **Once per N days** — Custom interval.
- **Always** — Shows every time the trigger condition is met.

**Animations**:

- **Entrance** — Fade, slide, zoom, or bounce in.
- **Exit** — Fade or slide out.

**Layout**:

- **Width** — Full screen, full width, or custom width.
- **Position** — Center, top, bottom, left, right.
- **Overlay** — Background overlay color and opacity.
- **Close button** — Show or hide, and customize its style.

## Set display conditions

Use the **Conditions** panel to control which pages show the popup:

- Entire site.
- Specific pages, posts, or products.
- Categories or tags.
- Custom URLs.

## Popup Close Button widget

When editing a popup with Elementor, you'll see a **Popup Close Button** widget in the panel. Drag it anywhere in your popup layout. You can style it as text, an icon, or an outlined button.

## Manage popups

Go to **Milano → Popups** to:

- See all popups in a list.
- Toggle them on and off.
- Reorder them (higher-priority popups display first).
- Edit or delete popups.

## Requirements

- Elementor (free version) must be installed and active.
