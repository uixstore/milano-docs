The Mega Menu module turns on support for rich, multi-column dropdowns in your navigation. When active, you can convert any menu item into a mega menu panel with columns, images, and widgets.

## Turn on Mega Menu

1. Go to **Milano Dashboard → Modules**.
2. Find **Mega Menu** and turn the toggle on.

The module is active by default after you install Milano.

## Set up your mega menu

Once the module is on, head to the Mega Menus guide for detailed setup instructions:
