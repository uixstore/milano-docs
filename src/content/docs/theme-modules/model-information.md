The Model Information module lets you create model profiles with measurements and assign them to products. Customers can see who's wearing the item and what size they're modeling.

## Turn on Model Information

1. Go to **Milano Dashboard → Modules**.
2. Find **Model Information** and turn the toggle on.

## Create a model profile

1. Go to **Model Profiles → Add New**.
2. Enter the model's name.
3. Fill in the measurements:
   - Height
   - Bust/Chest
   - Waist
   - Hips
   - Dress size / Pant size
   - Any custom fields you've defined
4. Add a photo of the model (optional).
5. **Publish** the profile.

## Assign a model to a product

1. Edit a product.
2. In the **Product Data** meta box, find the **Model Information** section.
3. Select one or more model profiles.
4. **Update** the product.

The model's name and measurements appear on the product page, typically near the size guide or in a separate section.

## Configure

You can customize which measurement fields appear and how they're labeled from **Model Profiles → Settings**.

## Requirements

- WooCommerce must be installed and active.
